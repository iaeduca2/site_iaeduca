# Diretrizes de Design - SENAI

## Paleta de Cores Oficiais
- Azul Principal: `#164194` (Uso em headers, botões primários, links principais)
- Laranja SENAI: `#E84910` (Uso em elementos de destaque, hover, badges de atenção)
- Cinza Fundo: `#F4F5F7` (Fundo das páginas)
- Branco: `#FFFFFF` (Fundo dos cards e modais)

## Tipografia (Web)
- Fonte Principal: Sans-serif limpa (Configurar Inter, Roboto ou Arial no Tailwind).
- Peso para Títulos: Bold / Semi-bold.

## Regras de Aplicação da Marca
- O logotipo deve sempre ter uma área de proteção (margem) mínima de 24px ao redor.
- Não utilizar o logo azul sobre fundos escuros (usar a versão branca/negativa se houver fundo escuro).
